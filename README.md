# CARGNet
we propose a novel weakly supervised change detection framework named CARGNet for point supervised change detection. 
# Dataset
## Dataset Structure
* `LEVIR-CD-Point`:
    * `train`:
      * `C`：Changed images.
      * `UC`：Unchanged images.
    * `test`:
      * `image`
      * `image2`
      * `label`
* `DSIFN-CD-Point`:
    * `train`:
      * `C`：Changed images.
      * `UC`：Unchanged images.
    * `test`:
      * `image`
      * `image2`
      * `label`
